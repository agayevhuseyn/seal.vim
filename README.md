# seal.vim
Seal syntax plugin for Vim

## Installation
For `packer.nvim`, add this to the `packer` `startup` function:

```lua
use('agayevhuseyn/seal.vim')
```
For Vim, run this:
    
    git clone https://github.com/agayevhuseyn/seal.vim ~/.vim/pack/seal/start/seal.vim
